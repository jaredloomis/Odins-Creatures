# Odin's Creatures

A Dwarf Fortress type game in Rust using Bevy. The goal is to create a game with interesting emergent complexity with
relatively minimal code.

The plan is to have a simple 2D GUI using [Bevy](bevy.org).

## Roadmap

- [] 2D GUI with dwarves
- [] Command palette
- [] Mining
- [] Logging, trees, building materials, building ability for dwarves
- [] Plants, harvesting, food, hunger, death
- [] Water, thirst, dehydration
- [] World generation
- [] Combat, Orcs/Elves/etc.
- [] Crafting
